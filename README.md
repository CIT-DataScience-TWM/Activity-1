# Activity-1

## Overview
This repository is created for the students of CIT M.Sc Data Science to submit their Activity-1 work.

## Instructions

1. Create a folder using your Roll Number or Name.
2. Upload all files related to Activity-1 inside your folder.
3. Do not modify, delete, or rename other students' folders or files.
4. Commit your changes with a meaningful commit message.
5. Ensure all required files are uploaded before the submission deadline.

## Folder Structure

```text
Activity-1
│
├── RollNo_1/
│   ├── file1
│   └── file2
│
├── RollNo_2/
│   ├── file1
│   └── file2
│
└── RollNo_3/
    ├── file1
    └── file2
```

## Naming Convention

Use your roll number as the folder name.

Example:

2503717673821001
2503717673821036
................

## Submission Steps

1. Open the repository.
2. Create your folder.
3. Upload your files.
4. Commit changes.
5. Verify that your files are visible in the repository.

## Important Notes

- Upload files only inside your own folder.
- Do not edit another student's work.
- Ensure files are uploaded correctly before the deadline.
- Contact the faculty/coordinator if you face any GitHub issues.
